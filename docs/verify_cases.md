## 1. 功能模块概述

### 1.1 核心功能模块

| 模块 | 功能 | 路由 |
|------|------|------|
| 首页 | 今日统计、本周概览、快速开始运动 | `/` |
| 运动记录 | 运动历史列表、详情查看 | `/exercise`, `/exercise/:id` |
| 添加运动 | 手动添加运动记录 | `/exercise/add` |
| 实时运动 | GPS实时追踪运动 | `/exercise/active` |
| 数据统计 | 7天/30天数据筛选 + 图表 | `/statistics` |
| 身体指标 | BMI/BMR计算展示 | `/body` |
| 身体录入 | 身体数据录入 | `/body/input` |
| 设置 | 主题切换 | `/settings` |

### 1.2 运动类型

| 类型 | 名称 | 是否需要GPS | 图标 |
|------|------|------------|------|
| running | 跑步 | 是 | 🏃 |
| cycling | 骑行 | 是 | 🚴 |
| walking | 健走 | 是 | 🚶 |
| gym | 健身 | 否 | 💪 |

## 2. 测试环境

### 2.1 设备要求

- **测试设备**: Android 模拟器 (API 36) / Android 真机
- **系统版本**: Android 7.0 (API 24) 及以上
- **屏幕分辨率**: 1080x2400 (模拟器)

### 2.2 测试账号

- 无需登录，数据存储在本地SQLite数据库

### 2.3 测试前准备

1. 确保应用已安装 (`app-debug.apk` 或 `app-release.apk`)
2. 模拟器/真机已连接并正常运行
3. 开启GPS定位权限
4. 准备测试数据

## 7. 附录

### 7.1 BMI计算公式

```
BMI = 体重(kg) / 身高(m)²

分类标准:
- < 18.5: 偏瘦
- 18.5 ~ 23.9: 正常
- 24 ~ 27.9: 超重
- >= 28: 肥胖
```

### 7.2 BMR计算公式（ Mifflin-St Jeor ）

```
男性: BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 + 5
女性: BMR = 10×体重(kg) + 6.25×身高(cm) - 5×年龄 - 161
```

### 7.3 卡路里计算公式

```
卡路里 = MET值 × 体重(kg) × 运动时长(小时)

常见运动MET值:
- 跑步: 9.8
- 骑行: 7.5
- 健走: 3.8
- 健身: 6.0
```
